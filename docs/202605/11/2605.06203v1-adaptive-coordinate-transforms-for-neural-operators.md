## 摘要
神经算子在偏微分方程（PDE）求解方面取得了显著进展，但大多数现有模型建立在固定的欧拉坐标系上。演化的物理结构与静态坐标之间的这种不匹配导致了空间错位，从而产生了不必要的非局部算子映射，并强化了在剧烈过渡区域附近的平滑偏好。受经典 PDE 分析中自适应坐标变换的启发，我们提出了自适应坐标变换（ACT）块，这是一种用于神经算子中数据驱动几何自适应的即插即用模块。ACT 块通过在算子学习流程中学习自适应坐标系来解决这一结构性限制。具体而言，给定输入特征，ACT 块学习一种坐标变换，并通过可微采样在变换后的坐标下表示相同的特征。该操作在改变空间表示的同时保留了底层信号，相当于在不同的坐标系中表达相同的物理量。通过使坐标系适应数据，ACT 允许网络更好地追踪演化结构，降低算子复杂度，并动态关注关键特征以提高学习效果。我们在多种 PDE 基准测试和多种神经算子架构上评估了所提方法。实验结果表明，预测精度得到了持续且显著的提升，这表明学习坐标系为增强算子学习提供了一种强大的机制。

## Abstract
Neural operators have achieved promising performance on partial differential equations (PDEs), but most existing models are built on fixed Eulerian coordinates. This mismatch between evolving physical structures and static coordinates creates spatial misalignment, leading to unnecessarily non-local operator mappings and reinforcing a smoothness preference near sharp transitions. Inspired by adaptive coordinate transformations in classical PDE analysis, we propose the Adaptive Coordinate Transform (ACT) block, a plug-and-play module for data-driven geometric adaptation in neural operators. ACT blocks resolve this structural limitation by learning adaptive coordinate systems within the operator learning pipeline. Specifically, given an input feature, the ACT block learns a coordinate transformation and represents the same feature under the transformed coordinates via differentiable sampling. This operation preserves the underlying signal while changing its spatial representation, equivalent to expressing the same physical quantity in different coordinate systems. By adapting the coordinate system to the data, ACT allows the network to better track evolving structures, reduce operator complexity, and dynamically focus on critical features to improve learning. We evaluate the proposed approach across diverse PDE benchmarks and multiple neural operator architectures. Experimental results demonstrate consistent and significant improvements in predictive accuracy, indicating that learning coordinate systems provides a powerful mechanism for enhancing operator learning.

### 1. 核心问题与研究动机

*   **核心问题**：固定坐标系与随时间演化的物理结构（如移动的涡流、传播的波）之间存在天然的不匹配。这种“空间错位”导致算子映射变得极其复杂且非局部。此外，神经算子普遍存在“平滑偏好”，难以捕捉冲击波或剧烈过渡等高频物理现象。
*   **研究动机**：借鉴经典数值分析中的自适应网格（Moving Mesh）或 ALE 方法，通过让坐标系“追踪”物理特征，简化学习难度并提升对尖锐结构的捕捉能力。

### 2. 方法论：自适应坐标变换（ACT）块
*   **核心思想**：提出一种即插即用的 **ACT 模块**，它不改变物理信号本身，而是通过学习一个数据驱动的几何变换，在每一层动态调整特征的采样位置。
*   **关键技术细节**：
    *   **位移预测（Offset Prediction）**：利用轻量级的多尺度卷积模块，根据当前特征图预测每个像素点的坐标偏移量 $\Delta$。
    *   **可微采样（Grid Resampling）**：使用预测的偏移量对特征图进行重采样（双线性插值）。为了保持物理一致性，引入了**物理知情的边界扩展**（如周期性边界处理）。
    *   **残差连接与融合**：将变换后的特征与原始特征通过残差结构融合，确保训练稳定性。
    *   **层级化集成**：将 ACT 块交替插入到神经算子（如 FNO 的谱卷积层之间）中，实现连续的几何重对齐。
*   **角色解耦机制**：论文发现中间层的 ACT 负责追踪平流（对齐特征），而最后一层的 ACT 负责通过局部坐标压缩来放大梯度（恢复尖锐结构）。

    *   **Navier-Stokes**（流体动力学）、**Diff-Reaction**（反应扩散）、**Shallow Water**（浅水方程）、**Active Matter**（活性物质）、**TRL_2D**（湍流辐射层）、**MHD_64**（3D 磁流体力学）。
*   **基准模型（Baselines）**：对比了三种不同范式的神经算子：
    1.  **CNextU**（基于卷积的 ConvNeXt-UNet）。
    2.  **Transolver**（基于 Attention 的 Transformer 算子）。
    3.  **FNO**（基于频率域的傅里叶神经算子）。
*   **对比方式**：在完全相同的超参数和训练协议下，对比“原始模型”与“加入 ACT 模块后的模型”的 NRMSE（归一化均方根误差）。

### 4. 资源与算力
*   **硬件环境**：使用了配备 **4 张 NVIDIA A100 (80GB/40GB VRAM)** 的高性能计算集群。
*   **算力开销**：
    *   ACT 模块非常轻量。例如在 FNO 中，ACT 仅增加了 **0.27%** 的参数量。

    *   显存占用有所增加（如 CNextU 从 1.1GB 升至 4.2GB），主要是因为存储了重采样所需的网格信息。

### 5. 实验数量与充分性
*   **实验规模**：在 **6 个主流 PDE 数据集**上进行了全面测试，涵盖了从简单流体到复杂 3D 磁流体的场景。
*   **消融实验**：专门针对 FNO 进行了消融研究，对比了“增加参数量”、“仅在末层加 ACT”与“层级化加 ACT”的效果。
*   **充分性评价**：实验设计非常充分且公平。作者通过保持所有超参数一致，证明了性能提升并非来自参数量的简单堆叠，而是来自坐标变换这一归纳偏置。

### 6. 主要结论与发现
*   **显著提升精度**：ACT 在所有架构和数据集上均实现了性能提升。Transolver 的平均误差降低了 **76.4%**，FNO 降低了 **38.4%**，CNextU 降低了 **30.8%**。
*   **简化算子学习**：通过可视化证明，ACT 能够将复杂的平流运动在潜在空间中“拉直”，使后续的算子层只需处理更简单的残差动力学。
*   **克服平滑偏好**：末层的 ACT 块通过 Jacobian 乘子效应，能够显式放大空间梯度，从而精准还原物理场中的尖锐边缘。

### 7. 优点与亮点
*   **模型无关性**：作为即插即用模块，可适配卷积、Transformer 和谱算子等多种架构。
*   **极低参数开销**：以极小的参数代价换取了巨大的精度提升。
*   **物理启发**：将经典的数值分析思想（自适应网格）成功引入深度学习算子，具有很强的解释性。
*   **边界处理**：针对物理问题设计的周期性坐标折叠机制，有效避免了边界伪影。

### 8. 不足与局限
*   **适用范围限制**：对于本身不具备明显平流、变形或尖锐过渡的简单静态问题，ACT 的提升可能有限。
*   **理论深度**：虽然提供了直观的解释和公式推导，但对于“坐标变换如何降低算子复杂度”的严格数学证明仍有待加强。
*   **计算开销**：虽然参数量增加极少，但重采样操作（Grid Sample）会带来额外的显存占用和计算耗时，在大规模 3D 场景下可能面临压力。
*   **几何限制**：目前主要针对结构化网格（Cartesian grids），对于非结构化网格或极度复杂的几何边界，其适配性尚未充分讨论。
