## 摘要
傅里叶神经算子（Fourier Neural Operators）是学习函数空间之间映射的深度学习模型，可用于学习和求解偏微分方程（PDEs），在某些情况下其速度显著快于传统的 PDE 求解器。该模型内部包含傅里叶层，这些层直接对傅里叶模态应用线性变换，其参数取决于波数。然而，大多数物理系统是各向同性的，其结果与所选坐标系无关，但线性变换并不一定遵循这些对称性。我们对线性变换提出了一种改进，以确保遵循空间对称性，称之为各向同性傅里叶神经算子（Isotropic Fourier Neural Operator）。该方法不仅提高了模型性能，还将 2D 情况下的参数量减少了高达 16 倍，3D 情况下减少了高达 96 倍。

## Abstract
Fourier Neural Operators are deep learning models that learn mappings between function spaces and can be used to learn and solve partial differential equations (PDEs), in some cases significantly faster than traditional PDE solvers. Within the model are Fourier layers, which apply linear transformations directly to the Fourier modes, with parameters depending on the wave numbers. However, most physical systems are isotropic, with the results being independent of the coordinate system chosen, but the linear transformations do not necessarily respect these symmetries. We propose a modification to the linear transformations that ensures spatial symmetries are respected, called the Isotropic Fourier Neural Operator, which both improves model performance and reduces the number of parameters by up to a factor of 16 in 2D and 96 in 3D.

### 1. 论文的核心问题与整体含义

*   **核心问题**：大多数物理系统具有**各向同性（Isotropic）**，即物理规律不随坐标系的旋转或镜像而改变。然而，标准 FNO 的傅里叶核（Fourier Kernel）参数是独立学习的，并不保证遵循这些空间对称性。
*   **整体含义**：本文旨在通过在 FNO 的傅里叶层中引入空间对称性约束，开发出**各向同性傅里叶神经算子（Iso-FNO）**。这不仅能使模型符合物理直觉，还能大幅减少参数量并提升泛化性能。

### 2. 论文提出的方法论
*   **核心思想**：利用空间对称性（如反射、转置、旋转）来约束傅里叶核参数，使得模型输出随输入的变换而同步变换（等变性）。
*   **关键技术细节**：
    *   **对称性约束**：在 2D 空间中，要求傅里叶核 $R$ 遵循 $D_4$ 二面体群对称性。具体表现为：
        *   **x/y 轴反射**：$R_{k,l} = R_{-k,l}$ 且 $R_{k,l} = R_{k,-l}$。
        *   **坐标交换（转置）**：$R_{k,l} = R_{l,k}$。
    *   **实值约束**：为了保证实值输入得到实值输出，要求 $R_{k,l} = R^*_{-k,-l}$。结合上述对称性，这导致所有参数变为实数，进一步减半了参数量。
    *   **参数生成流程**：模型仅学习一个极小的上三角张量 $R_{iso}$，然后通过一系列反射和转置操作生成完整的傅里叶核 $R$。
*   **参数削减比例**：理论上，2D 场景下参数量减少至 1/16，3D 场景下减少至 1/96。
