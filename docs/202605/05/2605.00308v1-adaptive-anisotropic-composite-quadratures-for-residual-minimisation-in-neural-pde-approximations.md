## 摘要
我们研究了数值积分在神经网络逼近偏微分方程的残差最小化方法中的作用。首先，我们提出了一个抽象误差框架，将逼近误差、积分误差和优化误差分离开来，并推导出一个非线性 Strang 型估计，用于量化离散损失函数中的不准确性如何影响最终的逼近结果。受此分析启发，我们提出了一种各向异性自适应复合数值积分策略，该策略利用更丰富的参考积分和基于二分法的细化来控制残差损失的相对积分误差。随后，我们引入了一种基于刷新的训练方法，仅当在线误差指标超过预设阈值时才重新构建数值积分，从而平衡了精度和计算成本。在多项基准问题上的数值实验表明，所提方法缩小了训练损失与参考损失之间的差距，更高效地利用了积分点，并且相对于非自适应积分策略提供了更强的逼近精度。

## Abstract
We study the role of numerical quadrature in residual-minimisation methods for neural network approximation of partial differential equations. We first present an abstract error framework that separates approximation, quadrature and optimisation errors, and derive a nonlinear Strang-type estimate quantifying how inaccuracies in the discrete loss affect the final approximation. Motivated by this analysis, we propose an anisotropic adaptive composite quadrature strategy that controls the relative quadrature error of the residual loss using richer reference quadratures and bisection-based refinement. We then introduce a refresh-based training methodology that rebuilds the quadrature only when an online error indicator exceeds a prescribed threshold, balancing accuracy and computational cost. Numerical experiments on a range of benchmark problems show that the proposed approach narrows the gap between training and reference losses, uses quadrature points more efficiently and delivers strong approximation accuracy relative to non-adaptive quadrature strategies.

### 1. 论文的核心问题与整体含义
*   **研究背景**：在利用神经网络（NN）求解偏微分方程（PDE）时（如 PINNs 或深度最小二乘法），通常需要最小化物理残差的积分损失。
*   **核心问题**：现有的方法多采用蒙特卡洛（MC）或均匀配置点来近似积分。然而，如果积分点分布不当，优化器可能会在这些离散点上实现极低的损失（过拟合），而实际的连续残差依然很大。
*   **研究动机**：由于神经网络具有高度非线性和全局性，固定的积分方案难以捕捉训练过程中动态变化的局部特征（如边界层、激波）。论文旨在通过自适应数值积分（AQ）来显式控制积分误差，确保训练损失能真实反映逼近质量。

### 2. 论文提出的方法论
*   **理论框架**：提出了一个**非线性 Strang 型误差估计**，将总误差分解为逼近误差、积分误差和优化误差。证明了积分误差会作为乘法因子放大逼近误差，强调了控制相对积分误差的重要性。
*   **核心思想**：
    *   **各向异性 h-自适应复合积分**：将定义域划分为单元，在每个单元上同时使用“主积分规则”和更精确的“参考积分规则”。
    *   **误差估计器**：利用两套规则的差值作为局部误差指标 $\delta_K$。
    *   **各向异性细化**：采用基于二分法的细化策略。利用 Genz-Malik 启发式方法（计算被积函数的四阶差分）来决定在哪个轴向上进行分裂，从而以更少的点处理具有方向性的特征（如薄边界层）。
*   **算法流程（刷新机制）**：
    1.  初始化一个粗糙的积分网格。
    2.  **训练阶段**：在当前积分点上优化 NN 参数。
    3.  **监控阶段**：计算在线误差指标 $\eta$（训练损失与参考损失的相对差）。
    4.  **刷新阶段**：若 $\eta$ 超过预设阈值 $\tau$，则在 CPU 上触发自适应细化程序，重新构建积分点分布并传回 GPU，随后继续训练。

### 3. 实验设计
*   **实验场景**：
    *   1D/2D 对流扩散问题（高 Péclet 数，含强边界层）。
    *   (1+1)D 非线性方程：Burgers 方程（激波形成）、KdV 方程（孤子交互）、Cahn-Hilliard 方程（相分离）。
    *   2D 奇异性问题：L 型域 Poisson 问题、弧形波前问题。
    *   流体力学基准：2D Navier-Stokes（盖盖流，Re=3200）、2D Stokes（Moffatt 涡）。
    *   高维与参数化问题：3D 空间-时间热传导、参数化非线性反应扩散方程。
*   **对比基准（Benchmarks）**：
    *   **Uniform**：均匀网格复合 Gauss 积分。
    *   **Monte Carlo (MC)**：标准随机采样。
    *   **QMC (LHC/Halton)**：拉丁超立方采样和 Halton 序列。
*   **公平性处理**：基准方法的积分点总数被设定为自适应方法在整个训练历史中所用点数的 90% 分位数，以确保计算资源对比的公正性。

### 4. 资源与算力
*   **硬件**：使用单块 **NVIDIA V100 GPU** 进行训练。
*   **软件实现**：基于 Julia 语言（Flux.jl 用于神经网络，Gridap.jl 用于几何处理）。
*   **算力分配**：自适应积分的构建在 CPU 上完成，而损失函数计算和反向传播在 GPU 上完成。论文指出，AQ 的构建时间仅占总训练时间的一小部分，因为刷新频率较低。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了约 **12 组** 深入的基准实验，涵盖了从 1D 到 3D、从线性到非线性、从稳态到瞬态的多种 PDE 类型。
*   **充分性**：实验设计非常详尽，不仅对比了最终误差（$L^2$ 和 $H^1$ 误差），还展示了训练过程中“训练损失”与“参考损失”的演化曲线。
*   **客观性**：通过使用独立于训练点的精细网格进行验证，证明了自适应方法在消除“积分过拟合”方面的显著效果。

### 6. 主要结论与发现
*   **消除过拟合**：自适应积分能有效缩小训练损失与真实残差之间的差距，避免了非自适应方法中常见的“损失降、误差升”现象。
*   **高效率**：AQ 策略能自动将积分点集中在解变化剧烈的区域（如激波或奇异点附近），在相同精度下所需的积分点数远少于随机采样。
*   **鲁棒性**：对于高 Reynolds 数的流体问题或强对流问题，AQ 表现出极强的稳定性，而传统采样方法往往无法收敛到正确解。
*   **刷新频率**：实验发现，积分网格通常只需在训练初期刷新几次即可稳定，后期无需频繁更新。

### 7. 优点
*   **理论支撑**：不仅是启发式改进，还提供了非线性 Strang 估计的数学证明。
*   **各向异性优势**：相比各向同性细化，各向异性二分法极大地缓解了高维情况下的点数爆炸问题。
*   **通用性**：该框架可以无缝集成到现有的 PINNs 或其他残差最小化模型中。

### 8. 不足与局限
*   **实现复杂度**：相比于简单的随机采样，该方法需要复杂的几何网格处理逻辑，对开发者要求较高。
*   **高维限制**：虽然在 3D 表现良好，但对于极高维（如金融或量子力学中的数十维 PDE），基于网格的复合积分仍会面临维度灾难。
*   **CPU-GPU 通信**：频繁刷新积分点会引入数据传输开销，尽管论文中通过阈值控制降低了这一影响。
