## 摘要
将Transformer的位置编码适配到网格和图结构数据上带来了显著的计算挑战：精确的谱方法需要立方复杂度的特征分解，并且可能因数值求解器伪影而无意中破坏规范不变性；而高效的近似方法则在设计上牺牲了规范对称性。这两种失效模式都会在归纳学习中导致灾难性的泛化失败，即使用一组数值选择训练的模型，在遇到相似图的不同谱分解或同一网格的不同离散化时会失效。我们提出了GIST（规范不变谱Transformer），这是一种新的图Transformer架构，它通过随机投影实现端到端的$\mathcal{O}(N)$复杂度，同时通过对投影嵌入进行基于内积的注意力机制在算法上保持规范不变性，从而解决了这一挑战。我们证明了GIST能够以有界的失配误差实现离散化不变学习，使得参数能够在神经算子应用中跨任意网格分辨率进行迁移。实证结果表明，GIST在标准图基准测试中达到了最先进的性能（例如，在PPI数据集上达到99.50%的micro-F1），同时独特地扩展到具有多达75万个节点的基于网格的神经算子基准测试中，并在具有挑战性的DrivAerNet和DrivAerNet++数据集上实现了最先进的空气动力学预测。

## Abstract
Adapting transformer positional encoding to meshes and graph-structured data presents significant computational challenges: exact spectral methods require cubic-complexity eigendecomposition and can inadvertently break gauge invariance through numerical solver artifacts, while efficient approximate methods sacrifice gauge symmetry by design. Both failure modes cause catastrophic generalization in inductive learning, where models trained with one set of numerical choices fail when encountering different spectral decompositions of similar graphs or discretizations of the same mesh. We propose GIST (Gauge-Invariant Spectral Transformers), a new graph transformer architecture that resolves this challenge by achieving end-to-end $\mathcal{O}(N)$ complexity through random projections while algorithmically preserving gauge invariance via inner-product-based attention on the projected embeddings. We prove GIST achieves discretization-invariant learning with bounded mismatch error, enabling parameter transfer across arbitrary mesh resolutions for neural operator applications. Empirically, GIST matches state-of-the-art on standard graph benchmarks (e.g., achieving 99.50% micro-F1 on PPI) while uniquely scaling to mesh-based Neural Operator benchmarks with up to 750K nodes, achieving state-of-the-art aerodynamic prediction on the challenging DrivAerNet and DrivAerNet++ datasets.

## 1. 核心问题与整体含义
*   **研究动机**：将Transformer架构成功应用于图结构数据和网格（如计算流体力学中的表面网格）面临两大核心障碍：
    1.  **计算障碍**：精确的谱方法（如基于图拉普拉斯矩阵特征分解的位置编码）具有$\mathcal{O}(N^3)$的立方复杂度，无法扩展到大规模图。
    2.  **规范不变性障碍**：无论是精确还是近似的谱方法，都可能破坏“规范不变性”。这意味着模型训练时使用的特征向量方向、符号或排序等数值选择是任意的，当测试时遇到同一图的不同谱分解（例如不同求解器、不同网格分辨率）时，模型会因依赖于这些任意选择而出现灾难性的泛化失败。
*   **整体含义**：本文旨在设计一种同时解决计算复杂度和规范不变性问题的图Transformer架构，使其既能高效处理大规模图，又能作为**神经算子**，实现参数在不同离散化分辨率间的稳定迁移，这对于科学计算（如PDE求解）至关重要。

## 2. 提出的方法论
*   **核心思想**：通过限制注意力机制仅依赖于投影后嵌入之间的**内积**来算法性地保持规范不变性。虽然随机投影矩阵$R$本身破坏了规范对称性，但根据Johnson-Lindenstrauss引理，投影后嵌入的内积 $\langle R\phi_i, R\phi_j \rangle \approx \langle \phi_i, \phi_j \rangle$ 近似保持不变。因此，基于此内积计算的注意力权重将与具体的投影选择无关。
*   **关键技术细节与架构**：
    1.  **规范不变谱自注意力 (Gauge-Invariant Spectral Self-Attention)**：
        *   使用通过FastRP等快速近似方法得到的谱嵌入 $\tilde{\phi}_i = R\phi_i$ 作为查询(`query`)和键(`key`)。
        *   使用节点特征 $x_i$ 作为值(`value`)。
        *   注意力对数 $e_{ij} = \tilde{\phi}_i^\top \tilde{\phi}_j / \sqrt{d}$ ，仅依赖于内积，从而保持规范不变。
    2.  **规范等变谱自注意力 (Gauge-Equivariant Spectral Self-Attention)**：
        *   允许通过学习节点特征相关的标量来缩放谱嵌入 $\tilde{\phi}_i$ ，由于标量与正交投影可交换，此操作是“等变”的，不破坏不变性。
    3.  **多尺度架构 (Multi-Scale Architecture)**：
        *   受EfficientViT启发，采用三分支并行结构：
            *   **特征分支**：对节点特征的线性Transformer块。
            *   **局部分支**：图卷积层 + 线性Transformer块，强调局部信息。
            *   **全局分支**：`规范不变谱自注意力` + `规范等变谱自注意力` + 线性Transformer块，整合全局信息。
    4.  **线性复杂度实现**：
        *   结合线性注意力机制（如Katharopoulos等人提出的方法），将标准注意力的$\mathcal{O}(N^2d)$复杂度降至$\mathcal{O}(Nd^2)$。
        *   整体端到端复杂度为 $\mathcal{O}(N(d^2 + rk))$ ，其中 $N$为节点数，$d$为特征维度，$r$为投影维度，$k$为幂迭代次数。
