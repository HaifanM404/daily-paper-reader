## 摘要
无网格数值方法为复杂几何提供了灵活的离散化方案；然而，经典的无网格离散微分算子通常以牺牲计算成本换取有限精度，或以高昂的单点计算量换取高精度。我们引入了一个参数化框架，通过基于截断泰勒展开推导的多项式矩约束训练图神经网络，来学习无网格离散微分算子。该模型将局部模板的相对位置直接映射到离散算子的权重。当前工作表明，神经网络能够学习经典的多项式一致性，同时保持对不规则邻域几何的鲁棒性。所学习的算子仅依赖于局部几何结构，与分辨率无关，并且可以在不同的粒子配置和控制方程中重复使用。我们使用标准数值分析诊断方法评估该框架，结果显示其精度优于光滑粒子流体动力学方法，并在中等精度范围内相对于代表性的高阶一致性无网格方法具有更优的精度-成本权衡。通过使用学习到的算子求解弱可压缩纳维-斯托克斯方程，证明了其适用性。

## Abstract
Mesh-free numerical methods provide flexible discretisations for complex geometries; however, classical meshless discrete differential operators typically trade low computational cost for limited accuracy or high accuracy for substantial per-stencil computation. We introduce a parametrised framework for learning mesh-free discrete differential operators using a graph neural network trained via polynomial moment constraints derived from truncated Taylor expansions. The model maps local stencils relative positions directly to discrete operator weights. The current work demonstrates that neural networks can learn classical polynomial consistency while retaining robustness to irregular neighbourhood geometry. The learned operators depend only on local geometry, are resolution-agnostic, and can be reused across particle configurations and governing equations. We evaluate the framework using standard numerical analysis diagnostics, showing improved accuracy over Smoothed Particle Hydrodynamics, and a favourable accuracy-cost trade-off relative to a representative high-order consistent mesh-free method in the moderate-accuracy regime. Applicability is demonstrated by solving the weakly compressible Navier-Stokes equations using the learned operators.

## 2. 方法论：NeMDO框架
*   **核心思想**：将无网格离散微分算子的构造转化为一个学习问题。给定一个中心粒子及其邻域构成的局部模板图，使用图神经网络直接根据粒子的相对位置预测出用于近似目标微分算子（如梯度、拉普拉斯算子）的权重。
*   **关键技术细节**：
    1.  **问题形式化**：离散算子近似形式为 `L_D(φ(x_i)) = Σ_{j∈N_i} (φ(x_j) - φ(x_i)) * w_{ji}^D`。目标是学习映射 `f_θ: {相对位置} → {权重 w_{ji}^D}`。
    2.  **图结构构建**：以中心粒子为根节点，与其邻域粒子构成星形双向图。节点特征为归一化的相对位置向量。
    3.  **网络架构**：采用消息传递图神经网络。流程包括：相对位置编码MLP → L层图消息传递（使用基于交叉图匹配的注意力聚合）→ 输出MLP得到归一化权重。
    4.  **自监督训练目标**：关键创新点。不依赖任何PDE解数据或真实权重标签，而是通过强制满足多项式一致性条件进行训练。损失函数基于截断泰勒展开的多项式矩约束：
        `L = (1/N) Σ_i || Σ_j X̂_ji ŵ_{ji}^D(θ) - M_D ||_2^2`

    2.  **高阶与复杂算子**：探索更先进的网络架构和训练策略，以稳健地学习更高阶（p≥3）微分算子或矢量/张量算子。
    3.  **物理信息嵌入**：在训练中引入额外的物理约束（如守恒律、对称性），可能进一步提升算子的稳定性和物理保真度。
    4.  **硬件感知优化**：针对GPU等加速器进行端到端的架构和实现优化，并与传统方法在完整仿真流程中进行全面的成本-精度权衡分析。
    5.  **扩展到三维与复杂几何**：验证框架在三维问题和具有复杂边界几何的实际场景中的有效性与泛化能力。

## 10. 总结
本文提出的NeMDO框架是一项创新且扎实的研究。它成功地将图神经网络的自监督学习与无网格数值分析中的核心一致性原理相结合，创造了一个通用的离散微分算子的“神经构造器”。该方法在精度上超越了经典SPH核函数，并在中等精度需求下展现出比传统高精度一致方法更优的计算效率潜力。尽管存在对固定模板大小和高阶扩展等当前限制，但其系统性的验证、清晰的模块化设计以及对鲁棒性的关注，使其成为一个极具前景且可复用的基础工具。这项工作不仅推动了无网格计算方法的发展，也为科学机器学习中如何有效融合先验物理知识提供了有价值的范例。
